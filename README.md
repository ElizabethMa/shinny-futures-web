# shinny-futures-web-alpha

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

### Run your end-to-end tests
```
npm run test:e2e
```

### Run your unit tests
```
npm run test:unit
```

### publish
```
gh-pages -d dist
```
或者
```
git add dist/
git commit -m "xxx"
git subtree push --prefix dist origin gh-pages
```
